Catégorie : Apparence

Tous les blocs de cette catégorie affectent l'apparence du canevas (la zone où p5Visuel fait tous ses tracés) et des figures qui y sont tracées, que ce soit à l'aide de blocs de la catégorie Formes ou de blocs de la catégorie Tortue.


Fond

Paramètres
• une valeur couleur, obtenue à l'aide du bloc couleur ci-dessous

Fonctionnement
• efface le canevas en le remplissant avec la couleur spécifiée

Remarques
• les utilisateurs avancés peuvent aussi décrire la couleur dans un bloc texte (exemple : "red" ou "rgb(255,0,0)")


Trait

Paramètres
• une valeur couleur, obtenue à l'aide du bloc couleur ci-dessous

Fonctionnement
• à partir de maintenant, toutes les figures (formes ou tortue) seront tracés avec cette couleur

Remarques
• les utilisateurs avancés peuvent aussi décrire la couleur dans un bloc texte (exemple : "red" ou "rgb(255,0,0)")


Remplissage

Paramètres
• une valeur couleur, obtenue à l'aide du bloc couleur ci-dessous

Fonctionnement
• à partir de maintenant, le remplissage de toutes les figures (triangle, rectangle, ellipse, polygone) sera fait avec cette couleur

Remarques
• on peut aussi remplir les polygones tracés avec la tortue en les délimitant par les blocs
    Débuter le polygone et Terminer le polygone, disponibles dans la catégorie Formes
• les utilisateurs avancés peuvent aussi décrire la couleur dans un bloc texte (exemple : "red" ou "rgb(255,0,0)")


Couleur

Paramètres
• une valeur rouge (255 ci-dessus, mais qui peut être une expression numérique quelconque comprise entre 0 et 255)
• une valeur vert (255 ci-dessus, mais qui peut être une expression numérique quelconque comprise entre 0 et 255)
• une valeur bleu (255 ci-dessus, mais qui peut être une expression numérique quelconque comprise entre 0 et 255)
• une valeur opacité (255 ci-dessus, mais qui peut être une expression numérique quelconque comprise entre 0 et 255)

Fonctionnement
• spécifie une couleur dans le système RVBA

Remarques
• une valeur d'opacité de 255 rend la couleur complètement opaque
   une valeur d'opacité de 0 rend la couleur complètement transparente
• rappelons que le système RVB est décrit dans un article Wikipedia
• on peut expérimenter avec le système RVB en se rendant à la page web suivante


Choix de couleur

Paramètres : aucun

Fonctionnement
• permet de choisir une couleur dans une palette de 70 couleurs

Remarques
• s'applique aux formes et aux tracés tortue



Couleur choisie au hasard

Paramètres : aucun

Fonctionnement
• retourne une couleur choisie au hasard

Remarques
• s'applique aux formes et aux tracés tortue



Taille crayon

Paramètres
• une valeur trait (1 ci-dessus, mais qui peut être une expression numérique quelconque)

Fonctionnement
• spécifie l'épaisseur du trait qui sera utilisé pour tracer les figures subséquentes

Remarques
• s'applique aux formes et aux tracés tortue


Baisser crayon

Paramètres : aucun

Fonctionnement
• à partir de maintenant, la tortue va laisser une trace lors de ses déplacements

Remarques
• s'applique aux tracés tortue seulement


Lever crayon

Paramètres : aucun

Fonctionnement
• à partir de maintenant, la tortue ne laissera pas de trace lors de ses déplacements

Remarques
• s'applique aux tracés tortue seulement


Sans remplissage

Paramètres : aucun

Fonctionnement
• à partir de maintenant, les figures seront tracées sans remplissage

Remarques
• s'applique aux formes et aux tracés tortue


Rendre transparente la zone graphique courante

Paramètres : aucun

Fonctionnement
• Le fond de la zone graphique courante devient transparent.

Remarques
• s'applique aussi au canevas.


Fond de la page web

Paramètres
• une valeur couleur, obtenue à l'aide d'un des blocs couleur ci-dessus

Fonctionnement
• spécifie la couleur de fond de la oage web

Remarques
• les utilisateurs avancés peuvent aussi décrire la couleur dans un bloc texte (exemple : "red" ou "rgb(255,0,0)")
