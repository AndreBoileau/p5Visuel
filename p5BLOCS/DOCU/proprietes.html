Catégorie : Propriétés

Dans la catégorie précédente, on a vu comment créer des objets web. Et si on a pris la précaution de placer ces objets web dans des variables, on pourra consulter ou même changer certaines propriétés de ces objets à l'aide des blocs ci-dessous.


Valeur

Paramètres
• une variable objet

Fonctionnement
• retourne un nombre ou un texte, qui est le contenu de l'objet en question (entrée, glissière)


Contenu

Paramètres
• une variable objet

Fonctionnement
• retourne un texte, qui est le contenu de l'objet en question (texte, paragraphe, titre)


État de la case à cocher

Paramètres
• une variable objet

Fonctionnement
• retourne vrai si la case est cochée, et faux sinon


Cacher l'objet

Paramètres
• une variable objet

Fonctionnement
• rend invisible l'objet en question


Montrer l'objet

Paramètres
• une variable objet

Fonctionnement
• rend visible l'objet en question


Fixe la valeur

Paramètres
• une variable objet
• un variable valeur (0 ci-dessus, mais qui peut être un nombre ou un texte quelconque)

Fonctionnement
• le nombre ou le texte devient la valeur de l'objet en question (entrée, glissière)


Fixe le contenu

Paramètres
• une variable objet
• un variable valeur (vide ci-dessus, mais qui peut être un texte quelconque)

Fonctionnement
• le texte devient le contenu de l'objet en question (texte, paragraphe, titre)


Fixe le contenu d'une zone

Paramètres
• une variable objet
• un variable valeur (vide ci-dessus, mais qui peut être une liste quelconque)

Fonctionnement
• les éléments de la liste deviennent les lignes successives de la zone de texte


En cas de clic

Paramètres
• une variable objet
• une fonction action

Fonctionnement
• en cas de clic sur l'objet (bouton, canevas, etc) exécuter la fonction action


En cas de changement

Paramètres
• une variable objet
• une fonction action

Fonctionnement
• en cas de changement de l'objet (glissière, case à cocher) exécuter la fonction action


Parent de l'objet

Paramètres
• une variable objet1 (rien ci-dessus, mais qui peut être une variable quelconque)
• une variable objet2 (rien ci-dessus, mais qui peut être une variable quelconque)

Fonctionnement
• spécifie que l'objet1 sera contenu dans l'objet2
• quand notre programme s'exécute dans une page pré-définie, dans laquelle certains éléments ont été dotés d'un ID,
   l'objet2 peut être défini via le bloc « élément dont l'ID est »

Remarques
• réservé aux utilisateurs avancés


Style de l'objet

Paramètres
• une variable objet (rien ci-dessus, mais qui peut être une variable quelconque)
• une valeur style (vide ci-dessus, mais qui peut être un nom  quelconque)
• une valeur valeur (vide ci-dessus, mais qui peut être un nom  quelconque)

Fonctionnement
• spécifie la valeur du style de notre objet
• par exemple, spécifier que le style "font-size"  de notre paragraphe sera de "14pt"
• on peut trouver sur le web plus de renseignements sur les types/balises utilisable : ici par exemple

Remarques
• réservé aux utilisateurs avancés


Position de l'objet

Paramètres
• une variable objet (rien ci-dessus, mais qui peut être une variable quelconque)
• une valeur posX (0 ci-dessus, mais qui peut être un nombre  quelconque)
• une valeur posY (0 ci-dessus, mais qui peut être un nombre  quelconque)

Fonctionnement
• spécifie la position absolue de notre objet
• la position est spécifiée dans le système de coordonnées suivant :
    * l'origine (0,0) en en haut à gauche de la fenêtre
    * l'axe des x pointe vers la droite
    * l'axe des y pointe vers le bas
